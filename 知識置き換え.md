| AWS用語      | Tokiさんの知識で言うと               |
| ---------- | --------------------------- |
| EC2        | AWSで仮想サーバを作るサービス            |
| インスタンス     | VirtualBoxで作ったUbuntu仮想マシン1台 |
| AMI        | Ubuntu ISO・OSテンプレートみたいなもの   |
| キーペア       | SSH接続用の秘密鍵・公開鍵セット           |
| セキュリティグループ | AWS側のファイアウォール・ポート開放ルール      |
| VPC        | AWS内に作る自分専用ネットワーク           |
| サブネット      | VPCの中を区切った小さいネットワーク         |
| パブリックIP    | 外部からアクセスするための住所             |
