# EC2インスタンス作成

## 概要

AWS EC2でUbuntuサーバを作成した。

EC2はAWS上に作成できる仮想サーバであり、今回はWebサーバ構築の土台として使用した。

## 作成内容

- リージョン：東京リージョン
- AMI：Ubuntu Server
- インスタンスタイプ：無料利用枠対象の小さいインスタンス
- ストレージ：デフォルト設定
- キーペア：新規作成
- セキュリティグループ：SSHとHTTPを許可

## 手順

1. AWSマネジメントコンソールにIAMユーザーでログイン
2. EC2サービスを開く
3. 「インスタンスを起動」を選択
4. インスタンス名を設定
5. AMIでUbuntu Serverを選択
6. インスタンスタイプを選択
7. キーペアを作成
8. セキュリティグループを設定
9. インスタンスを起動

## 確認したこと

インスタンス起動後、以下を確認した。

- インスタンスの状態が `実行中` であること
- ステータスチェックに合格していること
- パブリックIPv4アドレスが割り当てられていること

## 学んだこと

EC2インスタンスを作成するには、OSとなるAMI、サーバの性能を決めるインスタンスタイプ、SSH接続に使うキーペア、通信を制御するセキュリティグループを設定する必要がある。

また、インスタンス作成後はパブリックIPv4アドレスを使ってSSH接続やブラウザアクセスを行う。
