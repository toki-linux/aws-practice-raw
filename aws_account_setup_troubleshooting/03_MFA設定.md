
# 3. MFA設定でQRコードが表示され、何をしているのかわからなかった
状況

ルートユーザーのMFA設定時にQRコードが表示された。

何を読み込めばよいのか、また認証アプリの数字がなぜ使えるのかがわからなかった。

## 原因

MFAの仕組みを理解していなかった。

## 対応

スマホの認証アプリでQRコードを読み取り、表示された6桁コードをAWSに入力してMFAを設定した。

## 学び

MFAは多要素認証のこと。

ログイン時に、

パスワード
認証アプリの6桁コード

の2つを使うことで、安全性を高める。

認証アプリはAWSと常に通信しているわけではなく、QRコードで共有した秘密情報と現在時刻をもとに6桁コードを計算している。

AWS側も同じ計算を行い、入力されたコードと一致すればログインを許可する。
