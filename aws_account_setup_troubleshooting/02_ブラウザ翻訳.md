
# 2. ブラウザ翻訳をONにしていると登録画面でエラーになった

### 状況

AWS登録画面で、請求情報や支払い情報を入力している途中にエラーが出た。

表示例：

```text
サーバーに問題があるため、リクエストを完了できませんでした。
```
## 原因

ブラウザの自動翻訳がAWSの入力フォームや画面処理に影響していた可能性がある。

## 対応

Chromeの翻訳をOFFにして、英語表示のまま進めたところ、登録処理が進んだ。

## 学び

AWSの登録画面や重要な設定画面では、ブラウザ翻訳が原因でフォーム処理がうまくいかない場合がある。

AWS操作時は、必要に応じて翻訳をOFFにして進める。
