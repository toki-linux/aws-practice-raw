
# 7. AWSコンソールを開いているだけで課金されるのか不安だった
## 状況

AWSコンソールのホーム画面を開いているだけで、料金が発生するのではないかと不安になった。

## 原因

AWSは従量課金であるため、どの操作が課金対象なのか理解できていなかった。

## 対応

ホーム画面を開く、請求画面を見る、サービス一覧を見るだけでは課金されないことを確認した。

## 学び

課金が発生するのは、主にリソースを作成・起動・保存・通信した場合。

例：

課金対象になり得るもの：
```txt
- EC2インスタンスの起動
- EBSボリュームの作成・保持
- Elastic IPの確保
- NAT Gatewayの作成
- S3へのデータ保存
- データ転送
```

基本的に課金されないもの：
- AWSコンソールにログインする
- ホーム画面を見る
- 請求画面を見る
- サービス一覧を見る
- Budgetsを作る
